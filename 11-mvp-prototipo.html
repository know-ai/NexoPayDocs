Pantallas

Las 10 Pantallas del MVP

Cada pantalla ha sido diseñada con precisión para cubrir el flujo completo del usuario, desde la primera interacción hasta la confirmación de pago exitoso.

🚀

1. Splash Screen

Pantalla de carga

Pantalla de carga con el logo animado de NexoPay. Transición fluida con efecto de desvanecimiento que transmite confianza y modernidad desde el primer segundo.

👋

2. Onboarding

Bienvenida

Carrusel de bienvenida con la propuesta de valor en 3 pasos. Ilustraciones claras que explican cómo NexoPay unifica todos los métodos de pago en una sola app.

🔐

3. Login

Autenticación

Autenticación con soporte para biometría (huella digital, Face ID), login social (Google, Apple) y credenciales tradicionales. Proceso seguro y rápido en menos de 5 segundos.

🏠

4. Home Dashboard

Panel principal

Balance total consolidado de todas las cuentas bancarias. Acciones rápidas (Pago Móvil, QR, PoS, Historial) y lista de transacciones recientes con actualizaciones en tiempo real.

💸

5. Pago Móvil

Transferencias

Envío de dinero con selector de contactos, selector de banco origen/destino, teclado numérico optimizado y confirmación instantánea. Compatible con todos los bancos del país.

📷

6. QR Code

Pagos instantáneos

Genera o escanea códigos QR para pagos instantáneos. Cámara integrada con detección automática, generación de códigos con monto y datos del comercio preconfigurados.

🏪

7. PoS Virtual

Comercios

Panel de comerciante con estadísticas diarias de ventas, generación de cobros vía QR, NFC o enlace compartido. Dashboard con métricas en tiempo real y reportes.

📋

8. Historial de Transacciones

Registro

Lista filtrable de todas las transacciones con agrupación por fecha. Filtros por tipo (envíos, recibidos, QR, PoS), rango de fechas y montos. Exportación a PDF disponible.

👤

9. Perfil

Configuración

Configuración de cuenta, seguridad (2FA, biometría), verificación KYC, gestión de cuentas bancarias vinculadas, preferencias de notificación y opciones de privacidad.

✅

10. Confirmación de Pago

Éxito

Animación de éxito con check animado, recibo digital completo con número de referencia, detalles de la transacción y opciones para compartir el comprobante.

Diseño

Principios de Diseño

Cada decisión de diseño en NexoPay está guiada por principios claros que priorizan la velocidad, la accesibilidad y la confianza del usuario.

🎨

Flutter Material Design 3

Implementación nativa de Material You con tokens de diseño dinámicos, componentes adaptativos y soporte completo para temas claro/oscuro.

📱

Mobile-First

Diseñado primero para móvil, donde ocurre el 98% del uso. Cada pixel optimizado para pantallas de 5" a 6.7" con interacciones táctiles nativas.

✨

UI Minimalista

Interfaz limpia y despejada que maximiza la usabilidad. Cada elemento tiene un propósito. Sin distracciones, sin elementos decorativos innecesarios.

♿

Accesibilidad Universal

Accesible para todos los perfiles demográficos: jóvenes digitales, adultos mayores, comerciantes y personas con discapacidades visuales. WCAG 2.1 AA compliant.

⚡

Sub-3 Segundos

Objetivo de completar cualquier pago en menos de 3 segundos. Optimización agresiva de cada paso del flujo para la máxima velocidad operativa.

👆

Bottom Navigation

Patrón de navegación inferior optimizado para uso con el pulgar. Acceso directo a las 4 funciones principales sin necesidad de alcanzar la parte superior.

Filosofía de diseño: "Cada toque cuenta." En mercados emergentes donde la conectividad puede ser inestable, el diseño debe ser resiliente, rápido y predecible. NexoPay minimiza los pasos necesarios y proporciona feedback instantáneo en cada interacción.

⚙️
State Management

Riverpod + BLoC pattern para un manejo de estado predecible, testeable y escalable con separación clara de responsabilidades.

🧭
Navegación

GoRouter para navegación declarativa con deep linking, route guards para autenticación y transiciones animadas personalizadas.

🔌
API Layer

Dio + Retrofit para comunicación HTTP con interceptores de autenticación, retry logic y cache offline-first.

💾
Local Storage

Hive para datos estructurados, Flutter Secure Storage para tokens y credenciales, SharedPreferences para configuración.

🧪
Testing

Unit tests con Mockito, widget tests con WidgetTester, integration tests con Patrol. Cobertura objetivo: 85%+.
