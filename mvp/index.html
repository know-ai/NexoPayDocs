Resumen de Términos de Servicio

NexoPay opera como plataforma de pagos interoperable. Las transacciones son irreversibles una vez confirmadas por la red bancaria. El usuario acepta el uso de autenticación reforzada (PIN y biometría) para operaciones sensibles.

Resumen de Política de Privacidad

Tus datos son cifrados en tránsito y reposo. No vendemos información personal. Compartimos solo datos estrictamente necesarios con bancos aliados y entes regulatorios para cumplimiento KYC/AML.
